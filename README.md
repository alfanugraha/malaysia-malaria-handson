
# Malaysia Malaria (hands on session)

<!-- badges: start -->
<!-- badges: end -->

The goal of this project is to predict the number of malaria cases in Malaysia using a dataset that contains information about malaria cases in Malaysia from 2000 to 2020. The dataset includes the following variables:

- Year: The year of the malaria cases
- State: The state in Malaysia where the malaria cases were reported
- Cases: The number of malaria cases reported in that year and state
- Population: The population of the state in that year
- Incidence Rate: The number of malaria cases per 100,000 population
